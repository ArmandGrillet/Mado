Q&A

What is Mado?

Mado is a Markdown editor application. It works on Linux, Mac OS X and Windows. You can use it to take notes, write blog posts or edit documents.

Does Mado work offline?

Yes, it does. Mado was designed to work offline just fine.

What does Mado know about me?

Mado knows things like how many users are using it but it never analyzes your documents.
