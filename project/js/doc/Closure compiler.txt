Il faut utiliser les @externs_url pour empêcher le renommage de certaiens fonctions.

Exemple:
// ==ClosureCompiler==
// @compilation_level ADVANCED_OPTIMIZATIONS
// @externs_url http://closure-compiler.googlecode.com/git/contrib/externs/chrome_extensions.js
// ==/ClosureCompiler==

Si l'on veut empêcher les problèmes avec les fonctions jQuery et de plugins jQuery le mieux c'est
// @externs_url http://closure-compiler.googlecode.com/git/contrib/externs/jquery-1.9.js
